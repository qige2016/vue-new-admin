# 打包

当项目开发完毕，只需要运行一行命令就可以打包你的应用：

npm run build

# 分析打包文件体积

如果你的打包文件很大，你可以通过 report 命令打包并分析依赖模块的体积分布，从而优化你的代码：

npm run report
